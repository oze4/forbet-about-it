#### Firefox add-on to remove annoying 'disable-ad-block-to-continue' overlay on Forbes.com

Got sick of manually removing HTML for those 'disable your ad-block' overlays, so I wrote this.

Right now, it only works on Forbes, but I plan on adding to it and keeping it updated.

# Install

Since it is not listed on AOM **but still signed by AOM**, you can download `forbet_about_it.xpi`, then double click it
